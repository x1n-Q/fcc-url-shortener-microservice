freeCodeCamp API Project

URL Shortener Microservice

Submit a valid HTTP or HTTPS URL and receive a short numeric ID you can use for redirects.

Example POST

[project_url]/api/shorturl - body (urlencoded)
url=https://www.freecodecamp.org

Example Response

{ "original_url": "https://www.freecodecamp.org", "short_url": 1 }
